# Limpiador de Imágenes

Aplicación de escritorio **WPF** (.NET 8) para revisar y limpiar imágenes en carpetas.

## Requisitos

- **Windows** (interfaz WPF)
- [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)

## Ejecutar el proyecto

Desde la raíz del repositorio:

```bash
dotnet run --project src/LimpiadorImagenes/LimpiadorImagenes.csproj
```

En PowerShell también puedes usar barras invertidas:

```powershell
dotnet run --project src\LimpiadorImagenes\LimpiadorImagenes.csproj
```

Opcional: compilar y ejecutar el ensamblado generado en `src\LimpiadorImagenes\bin\Debug` o `Release` (por ejemplo `LimpiadorImagenes.exe`).

## Abrir en Visual Studio

Puedes abrir `LimpiadorImagenes.sln` y pulsar **Iniciar** (F5) con el proyecto `LimpiadorImagenes` como proyecto de inicio.
